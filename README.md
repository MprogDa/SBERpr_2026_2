# SBERpr_2026_2

## 1. User Story
### User
- Я как пользователь системы хочу регистрироваться и входить в систему, чтобы получить доступ к своим данным и рабочим пространствам.
- Я как пользователь системы хочу создавать и настраивать рабочие пространства.
- Я как пользователь системы хочу создавать задачи, назначать права, менять статусы, чтобы отслеживать прогресс работы.
- Я как пользователь системы хочу перемещать задачи между группами, чтобы визуально фиксировать изменение статуса.
- Я как пользователь системы хочу оставлять комментарии, прикреплять файлы к задачам.
- Я как пользователь системы хочу фильтровать и искать задачи.
### Admin
- Я как администратор системы хочу блокировать и разблокировать учетные записи пользователей.
- Я как администратор системы хочу просматривать системные логи и статистику использования.
- Я как администратор системы хочу удалять рабочие пространства.

## Требования 
### Функциональные требования
#### Управление пользователями и аутентификация
- Система должна позволять пользователям регистрироваться по email и паролю
- Система должна обеспечивать восстановление пароля через email
- Система должна обеспечивать аутентификацию пользователя 
- Система должна поддерживать управление профилем пользователя (логин, пароль, email).
- Система обеспечивает разделение ролей (Админ, пользователь)
#### Управление рабочими пространствами (досками)
- Система должна позволять создавать, редактировать и удалять рабочие пространства (доски)
- Система должна позволять редактировать доступ к  рабочим пространствам (доскам)
- Система должна позволять назначать роли участникам
- Система должна обеспечивать приватность (доступ только владельцу)
#### Управление задачами
- Внутри колонки пользователь может создать задачу
- Задачи отображаются внутри своих колонок
- Пользователь может редактировать и удалять задачи
- Возможноть использования разметки Markdown

### Нефункциональные требования
- Язык программирования: Java
- Основной фреймворк: Spring Boot
- Безопасность: Spring Security
- База данных: H2
- Сборщик проектов: Maven
- Для стилизации интерфейса используется Semantic UI
